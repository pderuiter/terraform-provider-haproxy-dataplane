# haproxy-dataplane_version (Data Source)

Use this data source to read runtime version metadata from HAProxy Data Plane API.

## Example Usage

Use this for environment checks and to gate configuration rollouts by API or HAProxy version.

```terraform
data "haproxy-dataplane_version" "current" {}

output "dataplane_version" {
  value = data.haproxy-dataplane_version.current.spec["api_version"]
}
```

You can combine this output with CI policy checks to prevent unsupported changes.
