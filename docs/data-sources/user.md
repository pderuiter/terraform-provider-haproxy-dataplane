# haproxy-dataplane_user (Data Source)

Use this data source to read existing user configuration from HAProxy.

## Example Usage

### Direct lookup

Use this pattern for a straightforward read of an existing object.

```terraform
data "haproxy-dataplane_user" "example" {
  userlist = "example"
}
```

### Lookup with module outputs

Use this pattern when a module consumes existing HAProxy objects and exports their identifiers.

```terraform
locals {
  user_lookup_name = "existing_user"
}

variable "userlist" {
  type = string
}

  userlist = var.userlist
}

  value = data.haproxy-dataplane_user.selected.id
}
```
