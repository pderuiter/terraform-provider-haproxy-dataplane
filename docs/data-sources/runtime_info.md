# haproxy-dataplane_runtime_info (Data Source)

Use this data source to read existing runtime_info configuration from HAProxy.

## Example Usage

### Direct lookup

Use this pattern for a straightforward read of an existing object.

```terraform
data "haproxy-dataplane_runtime_info" "example" {
}
```

### Lookup with module outputs

Use this pattern when a module consumes existing HAProxy objects and exports their identifiers.

```terraform
locals {
  runtime_info_lookup_name = "existing_runtime_info"
}

data "haproxy-dataplane_runtime_info" "selected" {
}

output "runtime_info_id" {
  value = data.haproxy-dataplane_runtime_info.selected.id
}
```
